Nashua Papercut BI Tool

This is a reporting tool that Nashua deployed with clients using Nashua printers to generate reports on how printers are being used, This tool provides clients with live Dashboard (Total print cost, cost per job type, total color pages printed, etc), Departmental reports, Management Reports, Managed Print Service Reports and scheduled Reports.

ECM

This is an Enterprise Content Management Application that Implements best Record management and bookkeeping practices. This application helps organisations create and update their business file plans, securely store their electronic documents on a structured folder , structure based on their created file plan and manage their record center. Captive Solutions and Namcol are using this product.

Social Security Commission BI Tool

This is a business Intelligence Dashboard for performance monitoring, that provide management with statistical insight of data capturing agents at SSC in real time, The tool is developed with node js, Angular, SQLite and MS SQL Server where the tool is pulling data.
